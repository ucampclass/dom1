# Instrucciones

 ```
1 Crea un archivo js con el nombre index.js. Este deberas agregarlo a tu archivo html

2 Debes implementar bootstrap en tu sitio web

3 Dentro de tu body, debes agregar las siguientes etiquetas

<section>
      <select class="form-select" aria-label="Default select example">
        <option selected>Selecciona</option>
        <option value="1">Uno</option>
        <option value="2">Dos</option>
        <option value="3">Tres</option>
      </select>
</section>

4 En tu js crea una constante con el nombre data y debe contener el siguinte arreglo de objetos.

const data = [
  { id: 4, name: 'Cuatro' },
  { id: 5, name: 'Cinco' },
  { id: 6, name: 'Seis' },
  { id: 7, name: 'Siete' },
];

5 Debes crear un nuevo option dentro del select. Estos nuevos valores se crearan a partir de la variable data. Debes recorrer el arreglo e ir creando nuevos options. El valor de option debe ser el valor del id, y para el caso del innerText debe ser el valor del name


```
